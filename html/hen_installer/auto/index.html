Supports CEX 4.80 - 4.82 / DEX 4.82 - 4.84 / CEX 4.83 - 4.91 HFW Firmware

INFO: If HEN is already installed, this will act as an Enabler

Instructions: Click the "Auto Install HEN" button. Wait.

* If initialization fails, set Home Page to about:blank and bookmark this page or return from history. Start browser again, load this page directly, and try again *
